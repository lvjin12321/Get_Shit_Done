## 🚀 作为 OpenClaw Skill 使用（推荐）

这是最简单的使用方式，**不用写代码，直接对话就能生成报告**。

### 使用方法

1. 把这个技能目录放入 OpenClaw 的 `skills/` 文件夹
2. 在对话中说："帮我写个报告"
3. 我会问你几个背景信息：

```
你好！我是报告老油条，帮你生成专业格式的职场报告。

请告诉我：
1. 做什么事？（周报/培训总结/项目复盘/...）
2. 给谁看？（直属领导/大老板/HR/...）
3. 有没有模板？（有就上传，没有我用标准格式）
4. 有什么成果或数据？（有就写，没有我帮你模糊处理）
```

4. 回答完后，我会生成一份：
   - ✅ **不吹牛**：数据可以模糊，但绝不编造
   - ✅ **能交差**：该有的部分都有，该有的流程都走
   - ✅ **不糊弄**：看起来专业，术语用得恰到好处
   - ✅ **格式完美**：严格按模板格式，无需二次加工

### 对话示例

```
你：帮我写个周报

我：好的，请问：
   1. 这周主要做了什么工作？
   2. 有没有模板格式要求？
   3. 需要突出哪些成果？

你：这周就摸了 5 天鱼，公司给了个 Excel 模板
   [上传模板.xlsx]

我：收到，5 分钟后给你一份格式完美的周报 🐾

[生成报告.xlsx] ← 直接交差
```

## ✨ 老油条心法

| 原则 | 说明 |
|------|------|
| 不吹牛 | 数据可以模糊，但绝不编造 |
| 能交差 | 该有的部分都有，该有的流程都走 |
| 不糊弄 | 看起来专业，术语用得恰到好处 |
| 有退路 | "持续优化""稳步推进""进一步完善" |
| 甩锅于无形 | "受限于外部因素""待跨部门协同后推进" |

## 📐 格式跟随原则

**给什么格式，输出什么格式：**

- 用户提供 **DOCX 模板** → 输出 DOCX 文档
- 用户提供 **XLSX 模板** → 输出 XLSX 表格
- 用户提供 **飞书云文档链接** → 输出飞书云文档
- 用户提供 **飞书智能表格链接** → 输出飞书智能表格
- 用户**没提供模板** → 询问或根据场景选择最合适格式

## 💻 独立脚本使用

如果你想直接运行 Python 脚本：

### 安装依赖

```bash
pip install python-docx openpyxl
```

### 生成标准周报

```bash
python scripts/generate_report.py --title "工作周报" --week "2026 年第 15 周" --output report.docx
```

### 生成学习报告（带模板）

```bash
python scripts/generate_study_report.py template.xlsx output.xlsx
```
